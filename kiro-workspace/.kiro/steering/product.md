# Product overview

## 제품 목적

Portfolio Lens는 토스증권 Open API의 계좌, 보유 종목, 체결 완료 주문, 종목·지수 일봉을 읽어 현재 포트폴리오, 일별 구성 변화, 위험·성과 분석, 전략 백테스트, AI 평가 보고서를 제공하는 개인용 읽기 전용 웹 대시보드다.

주요 사용자는 자신의 토스증권 계좌를 서버에서 직접 운영하면서 거래 기능 없이 포트폴리오 상태와 장기 성과를 살펴보려는 개인 투자자다.

## 제품 원칙

1. **조회 전용**: 주문 생성, 정정, 취소와 매수·매도 가능 수량 조회를 제공하지 않는다.
2. **서버 중심 보안**: 실제 토스증권·OpenAI·AWS 자격증명은 서버 환경에서만 사용한다.
3. **계산 근거 공개**: 입출금·예수금·배당 원장이 없는 경우 실제 계좌 전체 성과라고 주장하지 않고 보유주식 기반 추정치로 표시한다.
4. **국내·해외 통합**: 국내와 해외 종목을 같은 화면과 분석에서 함께 보여 주며, 필요한 경우 일별 USD/KRW 환율로 원화 환산한다.
5. **일 단위 기록**: 첫 체결일부터 현재까지 거래일과 비거래일을 포함한 KST 일별 시계열을 유지한다.
6. **일관된 시각 언어**: shadcn/ui 패턴의 다크 우선, 흑백 표면, 외곽선·그라데이션 없는 디자인을 사용하고 데이터 계열에만 구분 색상을 쓴다.
7. **재현 가능한 운영**: 로컬은 Docker Compose와 SQLite를 기본으로 하고, MySQL 연결 가능 시 자동 전환·마이그레이션한다.

## 최종 범위

- 비밀번호 로그인, 계좌 선택, 5초 단위 현재 보유자산 갱신
- 국내·해외 보유 종목, 평가액, 매입액, 평가손익, 일간손익
- 첫 체결일부터 일별 포트폴리오 복원과 종목별 평가액 누적 Area 차트
- 기간 프리셋과 달력 기반 사용자 지정 기간
- 종목 숨김 설정, 현재 미보유 과거 종목도 설정 목록에 포함
- 분석 탭의 포트폴리오 추정 OHLC 캔들, 벤치마크 비교, 위험·수익·집중도·기여도 지표
- 백테스트 탭의 현재 포트폴리오 불러오기, 국내·미국 종목 직접 구성, 현금흐름·리밸런싱·벤치마크 비교
- OpenAI Responses 호환 API를 통한 정형화된 평가 문구와 고정 React 보고서 템플릿
- 로컬 또는 비공개 S3 보고서 저장, 불투명 UUID 공개 링크
- 토스증권 경로와 호환되는 제한된 읽기 전용 API
- SQLite 기본 저장, 선택적 MySQL 사용과 SQLite → MySQL 자동 마이그레이션

## 명시적 제외 범위

- 주문 생성·정정·취소 및 자동매매
- 매수 가능 금액, 매도 가능 수량, 조건주문, 주문 추천
- 토스증권 WTS 거래내역 텍스트 붙여넣기 또는 HTML 파일 가져오기
- 제공되지 않는 입출금·예수금·배당 원장을 임의로 추정해 확정값처럼 표시하는 기능
- 과거 성과를 미래 수익 보장이나 매수·매도 조언으로 표현하는 기능
